# i2i-Academy-ContainerizationBasics-2

This project demonstrates containerization by deploying a simple static webpage using Docker and Docker Compose, both locally and on a Google Cloud Platform VM.

## Files
- `index.html` - Simple static webpage
- `docker-compose.yml` - Docker Compose configuration using nginx

## Tasks Completed
- Created a static webpage with a custom message
- Configured Docker Compose with nginx to serve the webpage on port 8080
- Deployed and tested locally via http://localhost:8080
- Deployed the same setup on a GCP VM and accessed via public IP
